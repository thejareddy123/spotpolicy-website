# SpotPolicy Website

A clean and responsive frontend website for **SpotPolicy** — a motor insurance service that helps users explore insurance options and connect directly via call or WhatsApp.

---

## Description

SpotPolicy is a frontend website built for a real motor insurance business. It presents the company's services in a professional layout, builds trust with visitors, and makes it easy for potential customers to get in touch instantly through call or WhatsApp buttons.

---

## Features

- Responsive design that works on mobile, tablet, and desktop
- Services section covering Car, Bike, and Commercial Vehicle Insurance
- Showcase of 22+ partner insurance companies with a sliding carousel
- One-click Call and WhatsApp contact buttons
- Multi-language support (English, Hindi, Telugu, Tamil, Kannada)
- Smooth scroll animations and hover effects
- Clean, modern UI with a professional look

---

---

## How to Run

No installation or setup required.

1. Download or clone this repository
2. Open the `spotpolicy/` folder
3. Double-click `index.html` to open it in your browser

That's it — the website runs entirely in the browser with no server needed.
